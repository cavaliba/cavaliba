## CAVALIBA (c) 2024


### Getting started

    git clone https://github.com/cavaliba/cavaliba.git
    cp env.template .env
    => edit/review .env, set passwords, URLs, secrets,  ...

    docker compose up 
    => point your browser to http://localhost[:port]
