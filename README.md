## CAVALIBA (c) 2024


### Getting started

    git clone https://github.com/cavaliba/cavaliba.git
    cp docker-compose.yml.template docker-compose.yml
    cp nginx.conf.template nginx.conf
    cp env.template .env
    => review .env

    docker compose up 
    => point your browser to http://localhost[:port]
